FocusShield

A Flutter mobile application that helps students evaluate and improve their study environment using onboard sensor data and external contextual information.

Watch Demo Video View Source Code

Project Overview

FocusShield explores how mobile sensing can support better study habits. The app combines motion sensing, simulated environmental indicators, local data storage, and contextual information to help users understand whether a study environment supports concentration.

The application demonstrates a complete mobile interaction flow: onboarding, starting a session, monitoring a live session, ending a session, reviewing results, checking history analytics, and viewing recommendations.

Quick Facts

  • Platform: Flutter mobile application
  • Course: CASA0015 Mobile Systems & Interactions
  • Theme: Connected Environment
  • Sensor used: Accelerometer
  • Data: Session history + charts + recommendations
  • Repository: GitHub source code + demo assets

Key Features

The project was designed as a coherent end-to-end prototype rather than a single isolated screen.

Sensor-driven interaction

Uses onboard sensing concepts to capture movement-related environmental conditions and turn them into a study quality narrative.

Session workflow

Supports a complete user flow from onboarding to start session, live session, end report, analytics, and recommendations.

Data visualisation

Includes score summaries, charts, and history views to help users reflect on repeated study sessions over time.
