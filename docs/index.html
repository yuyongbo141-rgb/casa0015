FocusShield

A Flutter mobile application that helps students understand how noise, light, movement, and environmental context affect study concentration.

Watch Demo Video View GitHub Repository
Problem: Students often know how long they study, but not how their environment affects focus. FocusShield turns environmental signals into a focus score, historical analytics, and personalised recommendations.

Key Features

Real-time Monitoring

Uses the phone accelerometer to detect movement and combines it with environmental indicators during study sessions.

Focus Scoring

Generates a composite focus score based on noise, movement, light, and session duration.

History & Recommendations

Stores sessions over time and turns them into trend views, environment comparisons, and actionable advice.

How It Works

User Journey

  • Splash and onboarding introduce the app and permissions
  • Dashboard summarises recent performance and trends
  • Start Session and Live Session monitor study conditions
  • End Report explains the final focus score
  • History, comparison, and recommendations support repeated use

Technology

  • Flutter mobile application
  • Accelerometer via sensors_plus
  • Local persistence via SharedPreferences
  • Charts via fl_chart
  • Weather context via OpenWeatherMap API
